# 前言

欢迎来到基于SSM（Spring、SpringMVC、MyBatis）的高校素质评价系统项目。本项目旨在为高校提供一个公平、公正、公开的素质评价平台，帮助学校更好地培养高素质人才。以下是对本项目的详细介绍。

## 内容介绍

本项目采用Java语言，结合Spring、SpringMVC、MyBatis等主流框架，以及JS、Vue、CSS3等前端技术进行开发。系统主要包括以下几个模块：学生信息管理、评价指标管理、评价流程管理、评价结果查询等。通过对学生综合素质的评价，为高校提供一个全面的、多维度的评价体系，促进学生全面发展。

## 技术介绍

- **语言：** Java
- **使用框架：** Spring、SpringMVC、MyBatis
- **前端技术：** JS、Vue、CSS3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12\14\16
